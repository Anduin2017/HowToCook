# 食品安全

## 中毒

以下食物有造成中毒的风险：

* 未成熟的青西红柿
* 未烹饪熟的四季豆
* 未煮熟的豆角
* 发芽的土豆
* 生豆浆
* 泡发时间过长的木耳
* ……（欢迎补充）

## 沙门氏菌感染

沙门氏菌较常见于动物源性食物，包括蔬菜也可能因受粪便污染而含有沙门氏菌。

下列食品有造成沙门氏菌感染的风险：

* 未完全煮熟的蛋
* 未完全煮熟的肉
* 未经过杀菌的奶

## 黄曲霉素

黄曲霉素常由黄曲霉及寄生曲霉等另外几种霉菌在霉变的谷物中产生，如大米、豆类、花生等，是目前为止最强的致癌物质。加热至 280℃以上才开始分解，所以一般的加热不易破坏其结构。

下列食品有造成黄曲霉素中毒的风险：

* 腐坏的花生
* 腐坏的大米
* 腐坏的玉米
