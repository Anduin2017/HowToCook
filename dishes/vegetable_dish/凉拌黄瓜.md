# 凉拌黄瓜的做法

预估烹饪难度：★  
预估卡路里：107 大卡  

## 必备原料和工具

- 黄瓜
- 醋
- 酱油
- 蒜

## 计算

每次制作前需确定份数（1份=1人份）：

| 原料 | 基础量 | 每份增量 |
|:---|:---|:---|
| 黄瓜 | 200克 | +200克 |
| 醋 | 7.5毫升 | +4毫升 |
| 酱油 | 5毫升 | +2.5毫升 |
| 蒜 | 3瓣 | +3瓣 |
| 盐 | 0.4克 | +0.2克 |
| 香油 | 5毫升 | +2毫升 |
| 蚝油 | 5毫升 | - |

## 操作

1. 黄瓜用刀拍扁，切3厘米长块，装碗备用。
2. 蒜拍碎切末，备用。
3. 碗中加少许白糖，与黄瓜拌匀，腌制15分钟（糖腌可提鲜保脆，避免出水变软）。
4. 依次加入醋、酱油、盐、蚝油、蒜末，搅拌均匀。
5. 淋入香油，再次拌匀即可。

## 附加内容

- 黄瓜两端若有苦味，切除后试用；建议去皮，口感更清爽。
- 现做现吃最佳；冷藏需用保鲜膜密封，8小时内食用完毕，避免过久腌制失去脆度。
